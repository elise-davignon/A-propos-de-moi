Les Éléments : à l'origine des mathématiques (automne 2018)

Liste des séances

Les séances sont données à compter de 19h à la Station Ho.st.

  1. 25 septembre 2018 : Introduction historique aux Éléments d'Euclide.
    Les maths dans l'antiquité; avant et après Euclide; Livre I (Définitions, postulats).
    • Notes de cours
  2. 9 octobre 2018 : La géométrie plane
    Livres I à IV avec grande emphase sur le livre I.
    • Notes de cours
    • Solutions de l'exercice 2
  3. 23 octobre 2018 : Nombres entiers, théorie des nombres, et introduction des ratios.
    Livres VII à IX, puis livres V et VI.
    • Notes de cours
    • Solutions de l'exercice 3
    • Le programme qui calcule les PGCD (exécutable Windows) et le code source (en C).
      Je vous jure que c'est pas un virus. Il se peut que Windows bloque automatiquement l'exécution du fichier GCD.exe. Si c'est le cas, l'option pour "exécuter quand même" est toujours quelque part.

      Pour exécuter le code sous Linux, téléchargez le fichier "main.c", ouvrir le terminal puis, dans le dossier où se trouve le fichier "main.c", exécuter les commandes gcc -o gcd main.c puis ./gcd. Si cela ne fonctionne pas, il faudra probablement installer GCC (GNU Compiler Collection) via votre gestionnaire de paquets.
  4. 6 novembre 2018 : Théorie des ratios, nombres irrationnels, géométrie avancée.
    Livre X sur les nombres irrationnels. Livres XI et XII sur la géométrie solide. Livre XIII sur les solides platoniciens.
    • Notes de cours
  5. 20 novembre 2018 : La géométrie à vapeur. Présentation de notions de géométrie moderne.
    • Notes de cours
